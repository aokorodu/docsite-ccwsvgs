## intro: shape elements

The shape elements provide the foundational visual building blocks of an SVG.

![shapes.svg](/shapes.svg)

In this section we'll discuss the basic shape elements and their attributes.

### presentation attributes

All shapes have presentation attributes that are used to define how they look. There are two types of presentation attributes:

**Geometric Attributes**:
These define the **position, size, and shape** of the shape element. Examples include x, y, width, height, cx, cy, r, etc. Each shape has it's own specific set of geometric attributes.

**"Look and Feel"**: Attributes:
These define the visual style of the SVG elements, such as color, stroke, fill, opacity, etc. These attributes are universal and can be used with any shape element.

<br />

In this section we'll focus on the **geometric attributes** for each shape. A more
detailed discussion of the universal "look and feel" presentation attributes can
be found <Link href={"/presentation-attributes"}>{"here"} </Link>.

<br />

### circle

- **cx, cy**: the x and y coordinates of the center of the circle
- **r**: radius

<Blocks>{`<svg width="500" height="500" viewbox="0 0 500 500">
  <circle cx="250" cy="250" r="200" fill="orange" />
</svg>`}</Blocks>

![shapes_basics.png](/shapes_circle.png)

---

### ellipse

- **cx, cy**: the x and y coordinates of the center of the ellipse
- **rx**: the radius of the ellipse along the x axis
- **ry**: the radius of the ellipse along the y axis

<Blocks>{`<svg width="500" height="500" viewbox="0 0 500 500">
  <ellipse cx="250" cy="250" rx="200" ry="100" fill="orange" />
</svg>`}</Blocks>

![shapes_ellipse.png](/shapes_ellipse.png)

---

### rectangle

- **x, y**: the x and y coordinates of the upper left corner of the rectangle
- **width**: width
- **height**: height

<Blocks>{`<svg width="500" height="500" viewbox="0 0 500 500">
  <rect x="50" y="50" width="400" height="400" fill="orange"></rect>
</svg>`}</Blocks>

![shapes_basics.png](/shapes_rect.png)

### rectangle (rounded)

- **x, y**: the x and y coordinates of the upper left hand cornder of the rectangle
- **width**: width
- **height**: height
- **rx, ry**: the radius of the circle that defines the corners

<Blocks>{`<svg width="500" height="500" viewbox="0 0 500 500">
  <rect
    x="50"
    y="50"
    width="400"
    height="400"
    rx="20"
    ry="20"
    fill="orange"
  ></rect>
</svg>`}</Blocks>

![shapes_basics.png](/shapes_round_rect.png)

### image

The image element allows you to add bitmap images - pngs or jpegs - to your svg.

- **href**: the image path
- **x** horizontal position
- **y** vertical position
- **width**: width
- **height**: height

<Blocks>{`<svg width="500" height="500" viewbox="0 0 500 500">
  <image 
      href="https://plus.unsplash.com/premium_vector-1724876723896..." 
      x="50" y="50" height="400" width="400" />
</svg>`}</Blocks>

<svg width="500" height="500" viewbox="0 0 500 500">
